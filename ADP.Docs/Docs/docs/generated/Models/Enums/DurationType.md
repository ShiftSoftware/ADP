Units of time duration used for validity periods, active durations, and other time-based configurations.

| Value | Summary |
|-------|---------|
| NotSpecified | No duration type specified. |
| Seconds | Duration measured in seconds. |
| Minutes | Duration measured in minutes. |
| Hours | Duration measured in hours. |
| Days | Duration measured in days. |
| Weeks | Duration measured in weeks. |
| Months | Duration measured in months. |
| Years | Duration measured in years. |
