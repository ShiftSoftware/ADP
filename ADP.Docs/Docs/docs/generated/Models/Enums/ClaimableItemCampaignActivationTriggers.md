Defines what event triggers a claimable item campaign to become active for a vehicle.

| Value | Summary |
|-------|---------|
| WarrantyActivation | The campaign activates when the vehicle's warranty is activated. |
| VehicleInspection | The campaign activates when a vehicle inspection is performed. |
| ManualVinEntry | The campaign activates when a per-VIN entry is manually recorded against the campaign
 (e.g., admin tags specific VINs as eligible for a one-off reward). |
