The manufacturer-side processing status of a warranty claim.

| Value | Summary |
|-------|---------|
| NA | No manufacturer status available (not applicable or not yet submitted). |
| Exported | The claim has been exported to the manufacturer's system. |
| Downloaded | The manufacturer has downloaded and acknowledged the claim. |
| Paid | The manufacturer has approved and paid the claim. |
| Rejected | The manufacturer has rejected the claim. |
| OnHold | The claim is on hold at the manufacturer's side, pending further review. |
