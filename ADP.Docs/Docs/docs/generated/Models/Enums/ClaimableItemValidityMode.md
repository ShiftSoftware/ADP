Determines how the validity period of a claimable item is calculated.

| Value | Summary |
|-------|---------|
| RelativeToActivation | Validity is calculated relative to the activation date (e.g., valid for X days after activation). |
| FixedDateRange | Validity uses fixed calendar dates regardless of when the item was activated (e.g., valid from date X to date Y). |
