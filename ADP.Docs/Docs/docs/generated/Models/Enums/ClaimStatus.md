The processing lifecycle status of a warranty or service claim.

| Value | Summary |
|-------|---------|
| Draft | The claim has been created but not yet submitted for processing. |
| PendingProcess | The claim has been submitted and is pending processing. |
| Accepted | The claim has been reviewed and accepted. |
| RejectedWithError | The claim was rejected due to an error that may be correctable. |
| RejectedPermanently | The claim was permanently rejected. |
| Certified | The claim has been certified by the manufacturer. |
| Invoiced | The claim has been invoiced for payment. |
