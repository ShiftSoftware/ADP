Represents a vehicle color in the broker system, with a code and description.
 Used for both exterior and interior colors on broker vehicles.

| Property | Summary |
|----------|---------|
| Code <div><strong>``string``</strong></div> | The color code. |
| Description <div><strong>``string``</strong></div> | A human-readable description of the color. |
