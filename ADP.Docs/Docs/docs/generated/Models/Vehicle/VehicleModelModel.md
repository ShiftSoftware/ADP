Represents a vehicle model variant with its full technical specifications.
 Each record defines a specific combination of model, variant, body type, engine, transmission, and other attributes for a brand.

| Property | Summary |
|----------|---------|
| BrandHashID <div><strong>``string``</strong></div> | The Brand Hash ID from the Identity System. |
| BasicModelCode <div><strong>``string``</strong></div> | The basic model code that identifies the vehicle model line. |
| SFX <div><strong>``string``</strong></div> | The suffix code differentiating trim or equipment levels within a model. |
| ModelCode <div><strong>``string``</strong></div> | The full model code identifying this vehicle model. |
| ModelDescription <div><strong>``string``</strong></div> | A human-readable description of the vehicle model. |
| VariantCode <div><strong>``string``</strong></div> | The variant code within the model range. |
| VariantDescription <div><strong>``string``</strong></div> | A human-readable description of the variant. |
| Katashiki <div><strong>``string``</strong></div> | The Katashiki (manufacturer-specific model identifier). |
| Class <div><strong>``string``</strong></div> | The vehicle class (e.g., Sedan, SUV, Truck). |
| BodyType <div><strong>``string``</strong></div> | The body type of the vehicle (e.g., Hatchback, Coupe, Wagon). |
| Engine <div><strong>``string``</strong></div> | The engine description (e.g., displacement, configuration). |
| Cylinders <div><strong>``string``</strong></div> | The number of cylinders in the engine. |
| LightHeavyType <div><strong>``string``</strong></div> | Whether this is a light or heavy vehicle type. |
| Doors <div><strong>``string``</strong></div> | The number of doors on the vehicle. |
| Fuel <div><strong>``string``</strong></div> | The fuel type (e.g., Gasoline, Diesel, Hybrid, Electric). |
| Transmission <div><strong>``string``</strong></div> | The transmission type (e.g., Automatic, Manual, CVT). |
| Side <div><strong>``string``</strong></div> | The steering/driving side (e.g., Left-Hand Drive, Right-Hand Drive). |
| EngineType <div><strong>``string``</strong></div> | The engine type classification. |
| TankCap <div><strong>``string``</strong></div> | The fuel tank capacity. |
| Style <div><strong>``string``</strong></div> | The style or design trim of the vehicle. |
