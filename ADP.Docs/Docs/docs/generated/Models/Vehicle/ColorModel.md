Represents a vehicle color option available for a brand.
 Each color has a code and a human-readable description.

| Property | Summary |
|----------|---------|
| BrandHashID <div><strong>``string``</strong></div> | The Brand Hash ID from the Identity System. |
| Code <div><strong>``string``</strong></div> | The color code used to identify this color in manufacturer systems. |
| Description <div><strong>``string``</strong></div> | A human-readable description of the color (e.g., "Pearl White", "Midnight Black"). |
