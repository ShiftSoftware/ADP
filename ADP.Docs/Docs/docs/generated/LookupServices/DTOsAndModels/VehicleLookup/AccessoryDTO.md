Represents an accessory installed on a vehicle, as returned by the vehicle lookup.

| Property | Summary |
|----------|---------|
| PartNumber <div><strong>``string``</strong></div> | The part number of the installed accessory. |
| Description <div><strong>``string``</strong></div> | A description of the accessory. |
| Image <div><strong>``string``</strong></div> | URL of an image of the installed accessory. |
