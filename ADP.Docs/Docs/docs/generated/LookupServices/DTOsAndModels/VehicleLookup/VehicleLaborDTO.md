Represents a labor line in a vehicle's service history.

| Property | Summary |
|----------|---------|
| LaborCode <div><strong>``string``</strong></div> | The labor operation code from the manufacturer's service catalog. |
| PackageCode <div><strong>``string``</strong></div> | The package code if this labor is part of a service package. |
| ServiceCode <div><strong>``string``</strong></div> | The distributor-level service code. |
| ServiceDescription <div><strong>``string``</strong></div> | A description of the labor operation performed. |
