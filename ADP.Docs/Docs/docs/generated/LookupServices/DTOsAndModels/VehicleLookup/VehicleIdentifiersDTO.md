Contains the core identification properties for a vehicle — VIN, variant code, Katashiki, and colors.

| Property | Summary |
|----------|---------|
| VIN <div><strong>``string``</strong></div> | The Vehicle Identification Number. |
| Variant <div><strong>``string``</strong></div> | The variant code string (encodes ModelCode + SFX + ModelYear). |
| Katashiki <div><strong>``string``</strong></div> | The Katashiki (manufacturer-specific model identifier). |
| Color <div><strong>``string``</strong></div> | The exterior color code of the vehicle. |
| Trim <div><strong>``string``</strong></div> | The interior trim/color code of the vehicle. |
| BrandID <div><strong>``string``</strong></div> | The Brand Hash ID from the Identity System. |
