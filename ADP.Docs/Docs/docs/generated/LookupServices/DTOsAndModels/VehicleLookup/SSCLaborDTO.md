| Property | Summary |
|----------|---------|
| LaborCode <div><strong>``string``</strong></div> | The labor operation code. |
| LaborDescription <div><strong>``string``</strong></div> | A description of the labor operation. |
| AllowedTime <div><strong>``decimal?``</strong></div> | The allowed time in hours for this labor operation. |
