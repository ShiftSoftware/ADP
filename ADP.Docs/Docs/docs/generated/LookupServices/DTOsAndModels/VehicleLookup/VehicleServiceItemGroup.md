Defines a grouping for service items, used to organize them into tabs in the UI.

| Property | Summary |
|----------|---------|
| Name <div><strong>``string``</strong></div> | The display name of the group/tab. |
| TabOrder <div><strong>``int``</strong></div> | The display order of this tab in the UI. |
| IsDefault <div><strong>``bool``</strong></div> | Whether this is the default (initially selected) tab. |
| IsSequential <div><strong>``bool``</strong></div> | Whether the service items in this group follow sequential validity (one after another). |
