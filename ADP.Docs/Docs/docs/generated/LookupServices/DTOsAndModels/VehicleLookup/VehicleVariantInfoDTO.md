Contains parsed variant information extracted from the vehicle's variant code string.

| Property | Summary |
|----------|---------|
| ModelCode <div><strong>``string``</strong></div> | The model code portion of the variant. |
| SFX <div><strong>``string``</strong></div> | The suffix code differentiating trim or equipment levels. |
| ModelYear <div><strong>``int?``</strong></div> | The model year extracted from the variant. |
