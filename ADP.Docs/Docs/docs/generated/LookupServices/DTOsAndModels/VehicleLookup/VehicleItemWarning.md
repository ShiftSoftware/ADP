Represents a warning message displayed to the user before claiming a service item.

| Property | Summary |
|----------|---------|
| Key <div><strong>``string``</strong></div> | A unique key identifying this warning type. |
| Title <div><strong>``string``</strong></div> | The warning title. |
| ImageUrl <div><strong>``string?``</strong></div> | An optional image URL displayed with the warning. |
| BodyContent <div><strong>``string``</strong></div> | The main body content/message of the warning. |
| ConfirmationText <div><strong>``string?``</strong></div> | Optional confirmation text the user must acknowledge to proceed. |
