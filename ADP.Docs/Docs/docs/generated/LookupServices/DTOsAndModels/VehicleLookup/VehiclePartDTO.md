Represents a part line in a vehicle's service history.

| Property | Summary |
|----------|---------|
| PartNumber <div><strong>``string``</strong></div> | The part number used in the service. |
| QTY <div><strong>``decimal?``</strong></div> | The quantity of this part used. |
| PackageCode <div><strong>``string``</strong></div> | The package code if this part is part of a service package. |
| PartDescription <div><strong>``string``</strong></div> | A description of the part. |
