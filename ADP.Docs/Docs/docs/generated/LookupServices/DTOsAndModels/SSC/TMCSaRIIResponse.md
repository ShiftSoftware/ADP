Response from the TMC SaRII (Safety Recall Information Interface) system for SSC lookups.

| Property | Summary |
|----------|---------|
| SSCLookupStatus <div><strong>``SSCLookupStatuses``</strong></div> | The result status of the SSC lookup. |
| TMCErrorCode <div><strong>``int``</strong></div> | The TMC error code if an error occurred (0 if no error). |
| ExceptionMessage <div><strong>``string``</strong></div> | The exception message if an error occurred. |
