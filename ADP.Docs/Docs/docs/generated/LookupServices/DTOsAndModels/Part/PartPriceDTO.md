Contains the pricing information for a part in a specific country and region.

| Property | Summary |
|----------|---------|
| CountryID <div><strong>``string``</strong></div> | The Country Hash ID. |
| CountryName <div><strong>``string``</strong></div> | The resolved country name. |
| RegionID <div><strong>``string``</strong></div> | The Region Hash ID. |
| RegionName <div><strong>``string``</strong></div> | The resolved region name. |
| RetailPrice <div><strong>``PriceDTO``</strong></div> | The retail price for this part in this region. |
| PurchasePrice <div><strong>``PriceDTO``</strong></div> | The retailer purchase price (distributor sell price) for this part. |
| WarrantyPrice <div><strong>``PriceDTO``</strong></div> | The warranty reimbursement price for this part. |
