Represents a Harmonized System (HS) code for a part in a specific country.

| Property | Summary |
|----------|---------|
| CountryIntegrationID <div><strong>``string``</strong></div> | The country's integration ID. |
| CountryName <div><strong>``string``</strong></div> | The resolved country name. |
| HSCode <div><strong>``string``</strong></div> | The HS code for this part in this country. |
