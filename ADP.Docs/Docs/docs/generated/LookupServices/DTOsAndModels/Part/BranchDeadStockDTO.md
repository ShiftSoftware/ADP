Represents dead stock quantity for a part at a specific branch.

| Property | Summary |
|----------|---------|
| CompanyBranchHashID <div><strong>``string``</strong></div> | The Branch Hash ID from the Identity System. |
| CompanyBranchName <div><strong>``string``</strong></div> | The resolved branch name. |
| Quantity <div><strong>``decimal``</strong></div> | The dead stock quantity at this branch. |
