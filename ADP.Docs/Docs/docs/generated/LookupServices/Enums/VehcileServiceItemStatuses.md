The status of a vehicle service item in the claiming lifecycle.

| Value | Summary |
|-------|---------|
| Processed | The service item has been claimed/processed. |
| Expired | The service item's validity period has expired. |
| Pending | The service item is available and waiting to be claimed. |
| Cancelled | The service item has been cancelled. |
| ActivationRequired | The service item requires warranty activation before it can be claimed. |
